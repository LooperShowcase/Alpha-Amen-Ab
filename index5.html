This is one of the more confusing things in coding. Functions are helpful if you want to run code multiple times without typing it multiple times, and it can be used to detect when things happen. This is the most
basic kind of function:

function My_Function()
print("This is my function!")
end

If you make a function like that, it will not actually run unless you "call" the function. If we wanted to call the function we made above, we would write this line of code right below it:

My_function

As you can see, we type the name of our function and make sure to put brackets.
