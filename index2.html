The first thing to learn in coding is printing. Printing is used to send messages and communicate.
This is helpful for t he game giving you information. To print, you have to type:
print()

The brackets are where you type what you want to print. If you enter just a word, it will not
work. You have to make it a string, by typing this symbol in between the brackets:
""
Between those symbols is where you type your word. So if I typed:
print("Hi")
It would print the word Hi.
