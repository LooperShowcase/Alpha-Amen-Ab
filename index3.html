One of the most useful things in coding are variables. Variables are used to assign values to words. Example:
Word = "Hello"
print(Word)

As you can see, instead of typing a string, we used our variable, which we assinged by putting a word, then an equal sign. So when we did "print(Word)" we would
print: "Hello".
